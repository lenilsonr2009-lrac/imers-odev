Descrição do Projeto: "Explorando a Bíblia"
Este é um projeto de uma página web interativa e dinâmica, criada para ser um portal de conhecimento bíblico. O site serve como uma ferramenta de estudo e devoção diária, oferecendo aos usuários acesso rápido a curiosidades, versículos e hinos, além de uma vasta coleção de informações bíblicas que podem ser pesquisadas, filtradas e ordenadas.

O site apresenta um conjunto rico de funcionalidades, divididas em três áreas principais:

1. Conteúdo Devocional Diário
Ao carregar a página, o usuário é recebido com três seções que mudam aleatoriamente a cada visita:

Curiosidade do Dia: Exibe uma curiosidade aleatória extraída do arquivo data.json.
Versículo do Dia: Apresenta um versículo bíblico selecionado aleatoriamente de uma lista pré-definida no código JavaScript.
Hino da Harpa Cristã: Mostra o título e um trecho de um hino da Harpa Cristã, também selecionado de forma aleatória.

2. Catálogo Principal de Curiosidades
O corpo principal do site é um catálogo de cartões, onde cada cartão representa uma curiosidade bíblica. Esta seção é totalmente interativa:

Busca Dinâmica: O usuário pode pesquisar por palavras-chave no nome, descrição ou referência das curiosidades. O resultado é exibido instantaneamente.
Filtro por Livro: Um menu suspenso permite filtrar as curiosidades por um livro específico da Bíblia. Este menu é preenchido dinamicamente com base nos livros presentes no arquivo data.json.
Ordenação Inteligente:
Por Nome: Organiza os cartões em ordem alfabética.
Por Referência Bíblica: Uma funcionalidade avançada que ordena os cartões seguindo a ordem canônica dos livros da Bíblia (de Gênesis a Apocalipse) e, em seguida, pelo capítulo, permitindo uma navegação lógica e sequencial.
Paginação: Para não sobrecarregar a tela, as curiosidades são divididas em páginas, com botões de navegação na parte inferior para facilitar o acesso a todo o conteúdo.

3. Experiência do Usuário
Carregamento Assíncrono: As curiosidades do arquivo data.json são carregadas em segundo plano. Enquanto isso, um "spinner" (ícone de carregamento) é exibido, melhorando a percepção de desempenho do site.
Links Externos: O rodapé oferece links úteis para recursos externos, como a Bíblia Online e a Harpa Cristã, enriquecendo a experiência de estudo do usuário.
Em resumo, "Explorando a Bíblia" é uma aplicação web moderna e bem estruturada que combina conteúdo informativo com ferramentas de interatividade poderosas, tornando o estudo da Bíblia mais acessível e engajador.
